# 이 챕터의 주제

툰 셰이딩
이 셰이더 실습을 통해서 행렬을 한 번에 여러개의 행렬을 미리 곱해둔 뒤 정점변환을 사용하는 방식이다.
하지만 우리는 디퓨즈(난반사)를 구할 때 월드 공간을 사용했던 것을 기억할 것이다. (왜냐면 빛의 위치가 월드공간 기준이였기 때문에..)
하지만 World, View, Projection 행렬들이 이미 다 하나로 합쳐졌기 때문에 행렬곱을 두번 해줘야한다는 안타까운 일이 발생한다.

그렇기에 사용하는 것이 바로 이 World 행렬의 역행렬(공간 변환 때 사용)을 이용해서 빛의 위치를 지역 공간으로 변환한 뒤
이를 사용하면 정점의 위치와 법선에 손을 대지 않고도 모든 배개변수들이 동일한 공간에 있을 수 있게 할 수 있다.
